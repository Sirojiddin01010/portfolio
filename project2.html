Website

I developed a website for a travel agency using Figma, HTML, CSS, and JavaScript. Using HTML, I structured the website with sections like the homepage, booking forms, and contact details. I added interactivity with JavaScript, including a booking form with validation and a destination image carousel. This project helped me refine my design and development skills, applying Figma, HTML, CSS, and JavaScript to create a functional and attractive website.
